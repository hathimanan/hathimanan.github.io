ChatGPT-4o Response Recycling / Hallucination

OpenAI ChatGPT-4o

When prompted to apply a second filter criterion, the model recycled its initial response verbatim rather than refining results. Root cause traced to the absence of RAG capability — the model answers entirely from memory, sourcing hallucinated data from unverified social media and informal news articles.

  • Trigger: Sequential multi-criteria filter in a single session
  • Root cause: No retrieval-augmented generation; pure memory recall
  • Impact: Misleading, unreliable output in multi-criteria queries
  • Published: Thought leadership article on Medium
